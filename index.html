Course Overview

What You Will Build & Learn

This 2-credit master's course covers the full AIoT stack — from IoT ecosystem fundamentals and machine learning theory to deploying quantized TinyML models on microcontrollers and connecting them to cloud platforms.

Module 1

IoT & AIoT Principles

Definitions, architecture, ecosystems, Industry 4.0 applications, and AIoT integration. Hands-on with Edge Impulse and hardware platforms.

Module 2

Machine Learning

ML/DL fundamentals, AI workflow, TensorFlow/Keras, responsible AI design, and feature engineering for embedded targets.

Module 3

Neural Networks

Perceptrons, MLP, CNNs, backprop, loss optimization, regularization, and model evaluation with real-world metrics.

Module 4

AI on IoT Systems

TinyML, Edge AI, TFLite Micro, model quantization (int8), EON™ Compiler, on-device deployment, and cloud telemetry integration.

Featured Project · 2026

Posture & Fatigue Monitor

AIoT-Based Posture and Fatigue Monitoring Using TinyML and Multimodal Sensors

Salcedo Valdivieso, Rafael A. · Barrios Guevara, Tomas

Two lightweight MobileNetV2 models run simultaneously on a single XIAO ESP32-S3 Sense to detect sitting posture (4 classes) and fatigue-related gestures (3 classes). An MPU6050 IMU provides inertial data fused with vision predictions via a weighted strategy (80/20). All inference runs locally at 200–250ms latency — no cloud required for detection.
