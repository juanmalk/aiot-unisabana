Course Syllabus

Syllabus & Weekly Schedule

5-week intensive master's course · 2 credits · 96 total hours (30h directed + 66h autonomous) · Universidad de La Sabana, Faculty of Engineering.

Universidad de La Sabana In-person Master's Program Edge Impulse Enterprise

Prerequisite

None

Open to master's students across engineering programs.

Evaluation

Project-Based

Application projects, rubrics, and oral/written presentations in English.

Hardware

XIAO ESP32-S3 Sense

+ Expansion Board + MPU6050. Programmed via Arduino IDE.

Duration

5 Intensive Weeks

One module per week, culminating in a live project defense.

Week Topic Labs & Resources
Week 1 — IoT & AIoT Principles Module 1
W01
IoT & AIoT Foundations
1.1 Definition, evolution, trends and challenges in IoT · 1.2 IoT Architecture & Ecosystem · 1.3 Application domains: Smart Cities, Industry 4.0 · 1.4 IoT + AI integration (AIoT) · 1.5 Hardware & software platforms: XIAO ESP32-S3 Sense, Edge Impulse, Ubidots
 Motion Recognition + Anomaly Detection with Smartphone
Week 2 — Machine Learning Fundamentals Module 2
W02
ML, Deep Learning & AI Workflow
2.1 Supervised / Unsupervised / Reinforcement Learning · 2.2 Deep Learning overview · 2.3 AI workflow: Data → Features → Model → Deploy · 2.4 TensorFlow / Keras introduction · 2.5 Responsible AI: bias, fairness, privacy
 Getting Started with XIAO ESP32-S3 Expansion Board Base Motion Recognition + Anomaly Detection with XIAO ESP32-S3
Week 3 — Neural Networks In Depth Module 3
W03
Perceptrons, CNNs & Training
3.1 The Perceptron · 3.2 Multi-Layer Perceptron (MLP) · 3.3 Convolutional Neural Networks: layers, pooling, feature maps, MobileNet · 3.4 Training: loss functions, Gradient Descent, Backpropagation · 3.5 Evaluation: accuracy, F1, ROC-AUC · 3.6 Regularization, dropout, data augmentation
 Keyword Spotting (KWS) Integration with ThingSpeak via WiFi
Week 4 — AI on IoT Systems (TinyML / Edge AI) Module 4
W04
TinyML, Quantization & On-Device Deployment
4.1 TinyML & Edge AI: why edge, latency, privacy, MCU constraints · 4.2 TensorFlow Lite Micro framework · 4.3 Model compression: INT8 quantization, pruning, EON™ Compiler · 4.4 Deploying to Arduino/ESP32: C++ library integration · IoT cloud connectivity: Ubidots, ThingSpeak, MQTT
Lab: Full pipeline on ESP32-S3
 Image Classification on XIAO ESP32-S3 Data Acquisition with SenseCraft AI
Week 5 — Final Project Presentations Capstone
W05
Integration, Defense & Demo
Project integration & end-to-end testing · IoT dashboard validation · Latency & accuracy benchmarking · IEEE-format paper submission (English) · Oral defense with live hardware demo · Peer evaluation · Course retrospective
Final Paper (IEEE) Oral Defense (EN) Live Demo

Assessment

Grading Breakdown

85%

Application Projects & Labs

Hands-on Edge Impulse labs, weekly project milestones, in-class exercises, and the final AIoT system deployment. Evaluated via rubric.

15%

Scientific Paper Analysis & Presentation

Analysis and oral presentation of a peer-reviewed scientific paper related to an AIoT use case. Assessed on depth of understanding, critical thinking, and clarity of communication.
